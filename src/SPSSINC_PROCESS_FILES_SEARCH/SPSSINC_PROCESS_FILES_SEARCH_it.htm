Cerca i casi in un insieme di file SAV SPSS

Questa procedura consente di cercare i casi in una raccolta di file di dati strutturati in maniera simile e di visualizzare le informazioni da quelle corrispondenti ai criteri di ricerca. È possibile specificare i file su cui eseguire la ricerca, il criterio di selezione dei casi e le variabili da visualizzare per i casi selezionati.

• Se il file attivo corrente non ha un nome di dataset, verrà chiuso quando verrà eseguita questa procedura.

File di dati di input specificati dall'espressione jolly Scegliere una cartella da elaborare e, facoltativamente, specificare un modello di file. Ad esempio,
c:\mydata\x*.sav
elabora tutti i file sav nella cartella c:\mydata i cui nomi iniziano con la lettera x.

• Se non viene fornita alcuna espressione nome-file, si presume il valore *.sav.

• È necessario scegliere questa opzione oppure la seguente opzione Dati di input, ma non entrambe.

File di dati di input specificati in un file. Elaborare tutti i file elencati nel file specificato. Il comando SPSSINC SPLIT DATASET può produrre un file nel formato corretto.

• È necessario scegliere questa opzione oppure l'opzione precedente Dati di input, ma non entrambe.

Espressione di ricerca. Immettere un'espressione che viene valutata come true o false utilizzando i valori delle variabili in ciascun caso. È equivalente a un'espressione immessa in Dati>Seleziona casi>Se.

• Se l'espressione non può essere valutata perché alcune variabili non esistono nel file oggetto della ricerca, il caso non viene selezionato.

• Utilizzare gli apici se occorre includere un valore stringa nell'espressione oppure utilizzare due virgolette. L'intera espressione sarà racchiusa tra virgolette.

Variabili da elencare. Selezionare una o più variabili da visualizzare per i casi corrispondenti.

Elaborazione del file di dati dopo ogni fase. Scegliere Chiudi per chiudere ogni dataset dopo che è stato sottoposto alla ricerca.

Funzioni aggiuntive

Eseguire il comando
SPSSINC PROCESS FILES /HELP.
per visualizzare la descrizione della sintassi completa. Questa finestra di dialogo implementa un sottoinsieme delle funzioni di SPSSINC PROCESS FILES. La finestra di dialogo Programmi di utilità>Elabora file di dati supporta le capacità generali di questo comando.

Per utilizzare un file per specificare i file che devono essere elaborati, creare un file con un singolo nome per riga, includendo il percorso del file. Il nome deve essere racchiuso tra virgolette ("). Quanto viene dopo sulla riga viene ignorato. Le righe vuote e le righe che iniziano con # vengono ignorate.

Requisiti

Questo comando richiede il plug-in Python Programmability. Il plug-in Python è disponibile sul CD di IBM SPSS Statistics CD oppure è possibile scaricarlo da SPSS Developer Central (www.spss.com/devcentral).
