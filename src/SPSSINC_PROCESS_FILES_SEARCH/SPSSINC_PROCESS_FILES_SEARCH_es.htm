Buscar los casos en un conjunto de archivos SAV de SPSS

Este procedimiento proporciona una forma para buscar a través de los casos en una colección de archivos de datos con una estructura similar y mostrar información de los que coincidan con los criterios de búsqueda. Especifique los archivos para buscar, el criterio de selección de casos y las variables para mostrar los casos seleccionados.

• Si el archivo activo actual no tiene un nombre de conjunto de datos, se cerrará cuando se ejecute este procedimiento.

Archivos de datos de entrada especificados por expresión comodín. Elija un directorio para procesar y, de forma opcional, especifique un patrón de archivo. Por ejemplo,
c:\mydata\x*.sav
procesaría todos los archivos sav en el directorio c:\mydata cuyos nombres empiecen con la letra x.

• Si no se proporciona ninguna expresión de nombre de archivo, se asume *.sav.

• Debe elegir esta opción o la siguiente opción Datos de entrada, pero no ambas.

Archivos de datos de entrada especificados en un archivo Procese todos los archivos listados e n el archivo especificado. El comando SPSSINC SPLIT DATASET puede generar un archivo en el formato correcto.

• Debe elegir esta opción o la opción Datos de entrada precedente, pero no ambas.

Expresión de búsqueda. Entre una expresión que se evalúe en verdadero o falso utilizando los valores de variables en cada caso. Es equivalente a una expresión que entraría en Datos>Seleccionar casos>Si.

• Si la expresión no se puede evaluar porque no existen algunas variables en el archivo en el que se busca, el caso no está seleccionado.

• Utilice comillas simples si debe incluir un valor de cadena en la expresión o utilice dos comillas dobles. (Toda la expresión estará rodeada por comillas dobles.)

Variables para listar. Seleccione una o más variables para que se visualicen para los casos de coincidencia.

Proceso de archivo de datos después de cada paso. Elija Cerrar para cerrar cada conjunto de datos después de que se busque.

Características adicionales

Ejecute el comando
SPSSINC PROCESS FILES /HELP.
para mostrar toda la descripción de la sintaxis. Este diálogo implementa un subconjunto de las características de SPSSINC PROCESS FILES. El diálogo Utilidades>Procesar archivos de datos soporte las funcionalidades generales de este comando.

Para poder utilizar un archivo para especificar los archivos que se deben procesar, cree un archivo con un nombre por línea, incluyendo la ruta del archivo. El nombre debe estar entre comillas dobles ("). Todo lo que aparezca detrás de la línea se ignora. Las líneas en blanco y las líneas que empiezan por # se ignoran.

Requisitos

Este comando requiere el complemento de programabilidad Python. Puede encontrar el complemento Python en el CD de IBM SPSS Statistics o puede descargarlo desde SPSS Developer Central (www.spss.com/devcentral).
