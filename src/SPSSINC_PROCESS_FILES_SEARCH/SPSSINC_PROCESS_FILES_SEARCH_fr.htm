Recherche des observations dans un ensemble de fichiers SAV SPSS

Cette procédure permet de rechercher parmi les observations dans une collection de fichiers de données de structure identique et d'afficher les informations de ceux correspondant aux critères de recherche. Indiquez les fichiers à rechercher, les critères de sélection des observations et les variables à afficher pour les observations sélectionnées.

• Si le fichier actif n'a pas de nom de jeu de données, il sera fermé lorsque cette procédure est exécutée.

Fichiers de données d'entrée indiqués par une expression générique : Choisissez un répertoire à traiter et, facultativement, indiquez un modèle de fichier. Par exemple,
c:\mydata\x*.sav
entraîne le traitement de tous les fichiers sav du répertoire c:\mydata dont les noms commencent par la lettre x.

• Si aucune expression de nom de fichier n'est fournie, l'expression *.sav est prise en compte.

• Vous devez choisir cette option ou l'option Données d'entrée suivante mais pas les deux.

Fichiers de données d'entrée indiqués dans un fichier : Traite tous les fichiers répertoriés dans le fichier indiqué. La commande SPSSINC SPLIT DATASET peut générer un fichier au format correct.

• Vous devez choisir entre cette option et l'option Données d'entrée précédente.

Expression de recherche : Entrez une expression évaluée sur true ou false à l'aide des valeurs de variable dans chaque observation. C'est une expression équivalente à celle qui serait entrée dans Données>Sélectionner observations>Si.

• Si l'expression d'évaluation ne peut pas être évaluée car certaines variables n'existent pas dans le fichier sur lequel porte la recherche, l'observation n'est pas sélectionnée.

• Utilisez des apostrophes si vous devez inclure une valeur de chaîne dans l'expression ou utilisez des guillemets. (L'intégralité de l'expression sera placée entre guillemets.)

Variables à répertorier : Sélectionnez une ou plusieurs variables à afficher pour les observations correspondantes.

Traitement des fichiers de données après chaque étape : Sélectionnez Fermer pour fermer chaque jeu de données une fois la recherche effectuée.

Fonctions supplémentaires

Exécutez la commande
SPSSINC PROCESS FILES /HELP.
pour afficher la description intégrale de la syntaxe. Cette boîte de dialogue met en oeuvre un sous-ensemble de fonctions de SPSSINC PROCESS FILES. La boîte de dialogue Utilitaires>Traiter les fichiers de données prend en charge les fonctionnalités globales de cette commande.

Pour utiliser un fichier en vue d'indiquer les fichiers à traiter, créez un fichier avec un nom par ligne et son chemin d'accès. Le nom doit être placé entre guillemets ("). Tout élément suivant sur la ligne est ignoré. Les lignes vides et celles commençant par # sont ignorées.

Conditions requises

Cette commande requiert le plug-in de programmabilité Python, fourni sur le CD-ROM IBM SPSS Statistics, ou téléchargeable à partir de SPSS Developer Central (www.spss.com/devcentral).
