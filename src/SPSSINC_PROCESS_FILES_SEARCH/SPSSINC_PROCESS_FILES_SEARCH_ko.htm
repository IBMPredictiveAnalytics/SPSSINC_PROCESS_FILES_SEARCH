SPSS SAV 파일 세트에서 케이스 검색

이 프로시저는 유사 구조의 데이터 파일 콜렉션에서 케이스를 검색하고 검색 기준에 맞는 케이스 정보를 표시하는 방법을 제공합니다. 검색할 파일, 케이스 선택 기준 및 선택한 케이스에 대해 표시할 변수를 지정합니다.

• 현재 활성 파일에 데이터 세트 이름이 없는 경우, 이 프로시저가 실행되면 닫혀집니다.

와일드카드 표현식으로 지정된 입력 데이터 파일. 처리할 디렉토리를 선택하고, 선택적으로 파일 패턴을 지정하십시오. 예를 들면 다음과 같습니다.
c:\mydata\x*.sav
이름이 x 자로 시작하는 c:\mydata 디렉토리에 있는 모든 sav 파일을 처리합니다.

• 파일 이름 표현식이 지정되지 않으면, *.sav로 간주합니다.

• 이 옵션 또는 다음 입력 데이터 옵션 중 하나(모두는 아님)를 선택해야 합니다.

파일에 지정된 입력 데이터 파일. 지정한 파일에 나열된 모든 파일을 처리합니다. SPSSINC SPLIT DATASET 명령은 올바른 형식의 파일을 생성할 수 있습니다.

• 이 옵션 또는 이전 입력 데이터 옵션(모두는 아님) 중 하나를 선택해야 합니다.

표현식 검색. 각 케이스에서 변수 값을 사용하여 True 또는 False로 평가되는 표현식을 입력하십시오. 이는 Data>Select Cases>If에 입력한 표현식과 같습니다.

• 검색 중인 파일에 일부 변수가 없으므로 표현식을 평가할 수 없는 경우 케이스가 선택되지 않습니다.

• 표현식에 문자열 값을 포함시켜야 하거나 두 개의 큰따옴표를 사용해야 하는 경우 작은따옴표를 사용하십시오. (전체 표현식은 큰따옴표로 묶입니다.)

나열할 변수. 매치하는 케이스에 대해 표시할 하나 이상의 변수를 선택하십시오.

각 단계 이후 데이터 파일 처리. 검색 후 닫기를 선택하여 각 데이터 세트를 닫으십시오.

추가 기능

다음 명령문을 실행하십시오.
SPSSINC PROCESS FILES /HELP.
전체 명령문 설명을 표시합니다. 이 대화 상자는 SPSSINC PROCESS FILES 기능의 부분집합을 구현합니다. 유틸리티>프로세스 데이터 파일 대화 상자는 이 명령의 일반 기능을 지원합니다.

파일을 사용하여 처리해야 하는 파일을 지정하려면, 파일 경로를 포함하여 라인당 하나의 이름으로 파일을 작성하십시오. 이름은 큰따옴표(")로 묶여야 합니다. 라인 다음 내용은 무시됩니다. 빈 라인과 #으로 시작하는 라인은 무시됩니다.

요구 사항
