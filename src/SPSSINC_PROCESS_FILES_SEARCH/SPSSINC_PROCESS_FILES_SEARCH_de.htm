Fälle in einem Set von SPSS-SAV-Dateien durchsuchen

Diese Prozedur bietet eine Möglichkeit, die Fälle in einer Sammlung ähnlich strukturierter Datendateien zu durchsuchen und Informationen aus den Fällen anzuzeigen, die den Suchkriterien entsprechen. Sie geben die zu durchsuchenden Dateien, das Kriterium zu Fallauswahl und die für die ausgewählten Fälle anzuzeigenden Variablen an.

• Wenn die derzeit aktive Datei keinen Datasetnamen hat, wird sie beim Ausführen dieser Prozedur geschlossen.

Durch Platzhalterausdruck angegebene Datendateien. Wählen Sie ein Verzeichnis für die Verarbeitung aus und geben Sie optional ein Dateimuster an. Beispiel:
c:\mydata\x*.sav
verarbeitet alle SAV-Dateien im Verzeichnis c:\mydata, deren Namen mit dem Buchstaben x beginnen.

• Wird kein Ausdruck für einen Dateinamen angegeben, wird *.sav angenommen.

• Sie müssen diese Option oder die folgende Eingabedatenoption auswählen. Die Optionen können jedoch nicht kombiniert werden.

In einer Datei angegebene Eingabedatendateien. Alle in der angegebenen Datei aufgelisteten Dateien werden verarbeitet. Der Befehl SPSSINC SPLIT DATASET kann eine Datei im korrekten Format erzeugen.

• Sie müssen diese Option oder die vorhergehende Eingabedatenoption auswählen. Die Optionen können jedoch nicht kombiniert werden.

Suchausdruck. Geben Sie einen Ausdruck ein, der unter Verwendung der Werte der Variablen in jedem Fall in 'true' oder 'false' ausgewertet wird. Er entspricht einem Ausdruck, den Sie in Daten>Fälle auswählen>Falls eingeben würden.

• Wenn der Ausdruck nicht ausgewertet werden kann, da einige Variablen in der gesuchten Datei nicht vorhanden sind, wird der Fall nicht ausgewählt.

• Verwenden Sie einfache Anführungszeichen, wenn Sie einen Zeichenfolgewert in den Ausdruck aufnehmen müssen, oder verwenden Sie zwei doppelte Anführungszeichen. (Der gesamte Ausdruck wird in doppelte Anführungszeichen eingeschlossen.)

Aufzulistende Variablen. Wählen Sie mindestens eine Variable aus, die für übereinstimmende Fälle angezeigt werden woll.

Datendateiverarbeitung nach jedem Schritt. Wählen Sie Schließen aus, um jedes Dataset zu schließen, nachdem es durchsucht wurde.

Weitere Funktionen

Führen Sie den folgenden Befehl aus:
SPSSINC PROCESS FILES /HELP.
Sie zeigen damit die vollständige Syntaxbeschreibung an. Dieser Dialog implementiert ein Subset der Funktionen von SPSSINC PROCESS FILES. Der Dialog Extras>Datendateien verarbeiten unterstützt die allgemeine Funktionalität dieses Befehls.

Wenn die zu verarbeitenden Dateien in einer Datei angegeben werden sollen, erstellen Sie eine Datei, die pro Zeile den Namen und den Pfad zu einer Datei enthält. Der Name muss in Anführungszeichen (") eingeschlossen sein. Der darauf folgende Text in der Zeile wird ignoriert. Leere Zeilen und mit # beginnende Zeilen werden ignoriert.

Anforderungen

Dieser Befehl erfordert das Python-Programmierungs-Plug-in. Sie finden das Python-Plug-in auf der IBM SPSS Statistics-CD oder Sie können es über SPSS Developer Central (www.spss.com/devcentral) herunterladen.
