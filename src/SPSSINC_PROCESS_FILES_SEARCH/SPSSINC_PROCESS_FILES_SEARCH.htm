Search the Cases in a Set of SPSS SAV Files

This procedure provides a way to search through the cases in a collection of similarly structured data files and display information from those matching the search criteria. You specify the files to search, the case selection criterion, and the variables to display for the selected cases.

• If the current active file does not have a dataset name, it will be closed when this procedure is run.

Input Data Files Specified by Wildcard Expression. Choose a directory to process and, optionally, specify a file pattern. For example,
c:\mydata\x*.sav
would process all the sav files in directory c:\mydata whose names start with the letter x.

• If no file-name expression is given,*.sav is assumed.

• You must choose either this option or the following Input Data option but not both.

Input Data Files Specified in a File. Process all the files listed in the specified file. The SPSSINC SPLIT DATASET command can produce a file in the correct format.

• You must choose either this option or the preceding Input Data option but not both.

Search Expression. Enter an expression that evalues to true or false using the values of variables in each case. It is equivalent to an expression that you would enter in Data>Select Cases>If.

• If the expression cannot be evaluated because some variables do not exist in the file being searched, the case is not selected.

• Use single quotes if you need to include a string value in the expression or use two double quotes. (The entire expression will be surrounded by double quotes.)

Variables to List. Select one or more variables to be displayed for matching cases.

Data File Processing after Each Step. Choose Close to close each dataset after it is searched.

Additional Features

Run the command
SPSSINC PROCESS FILES /HELP.
to display the entire syntax description. This dialog implements a subset of the features of SPSSINC PROCESS FILES. The Utilities>Process Data Files dialog supports the general capabilities of this command.

In order to use a file to specify the files that should be processed, create a file with one name per line including the path to the file. The name must be enclosed in double quotes ("). Anything following on the line is ignored. Blank lines and lines starting with # are ignored.

Requirements

This command requires the Python Programmability Plug-in, You can find the Python plug-in on the IBM SPSS Statistics CD, or you can download it from SPSS Developer Central (www.spss.com/devcentral).
