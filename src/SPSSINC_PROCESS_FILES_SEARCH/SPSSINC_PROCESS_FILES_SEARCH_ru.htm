Искать наблюдения в наборе файлов SAV SPSS

Эта процедура дает способ поиска по наблюдениям в собрании подобным образом структурированных данных и выводит информацию о данных, соответствующих критериям поиска. Вы задаете файлы для поиска, критерий отбора наблюдений и переменные вывода для отобранных наблюдений.

• Если у текущего активного файла нет имени набора данных, при запуске этой процедуры он будет закрыт.

Входные файлы данных, заданные выражением с символами подстановки. Выберите каталог для обработки и, если хотите, задайте паттерн файла. Например,
c:\mydata\x*.sav
будет указывать на обработку в каталоге c:\mydata всех файлов sav, имена которых начинаются с буквы x.

• Если выражение для имен файлов не приведено, будет подразумеваться, что задано *.sav.

• Необходимо выбрать или эту опцию, или следующую опцию Входные данные, но не обе опции одновременно.

Входные файлы данных, заданные в файле. Обработайте все файлы, перечисленные в заданном файле. Команда SPSSINC SPLIT DATASET может создать файл в правильном формате.

• Необходимо выбрать или эту опцию, или предыдущую опцию Входные данные, но не обе опции одновременно.

Выражение поиска. Выберите выражение, которое будет принимать значение true или false в зависимости от значений переменных для каждого наблюдения. Оно эквивалентно выражению, которое можно ввести через пункт меню Данные>Выбрать наблюдения>Если.

• Если значение логического выражения нельзя вычислить из-за отсутствия некоторых переменных в искомом файле, это наблюдение не выбирается.

• Используйте одинарные кавычки, если в выражение нужно включить строковое значение, или двойные кавычки. (Все выражение целиком будет заключено в двойные кавычки).

Переменные для списка. Выберите одну или несколько переменных для вывода соответствующих наблюдений.

Обработка файла данных после каждого шага. Выберите Закрыть, чтобы закрыть каждый набор данных после проведения в нем поиска.

Дополнительные возможности

Введите команду
SPSSINC PROCESS FILES /HELP.
чтобы вывести полное описание синтаксиса. В этом диалоговом окне реализовано подмножество возможностей команды SPSSINC PROCESS FILES. Диалоговое окно Утилиты>Обработать файлы данных поддерживает общие возможности этой команды.

Чтобы задать файлы для обработки с помощью одного файла, создайте файл, в котором каждая строка содержит одно имя, включающее в себя путь к файлу. Это имя должно быть заключено в двойные кавычки ("). Все остальное после имени в строке будет игнорироваться. Пустые строки и строки, начинающиеся с #, также игнорируются.

Технические требования

Этой команде требуется подключаемый модуль программирования Python. Его можно найти на установочном компакт-диске IBM SPSS Statistics или скачать с сайта SPSS Developer Central (www.spss.com/devcentral).
