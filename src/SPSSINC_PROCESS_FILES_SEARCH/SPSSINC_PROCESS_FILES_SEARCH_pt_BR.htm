Procurar os Casos em um Conjunto de Arquivos SPSS SAV

Este procedimento fornece uma maneira para procurar nos casos em uma coleção de arquivos de dados estruturados de forma semelhante e exibir informações a partir desses arquivos correspondentes aos critérios de procura. Você especifica os arquivos para procura, o critério de seleção de caso e as variáveis para exibir para os casos selecionados.

• Se o arquivo ativo atual não tiver um nome do conjunto de dados, ele será fechada quando este procedimento é executado.

Arquivos de Dados de Entrada Especificados pela Expressão Curinga. Escolha um diretório para processar e, opcionalmente, especifique um padrão de arquivo. Por exemplo:
c:\mydata\x*.sav
processaria todos os arquivos sav no diretório c:\mydata cujos nomes iniciarem com a letra x.

• Se nenhuma expressão de nome do arquivo for determinada, *.sav é assumido.

• Você deve escolher esta opção ou a seguinte opção Dados de Entrada, mas não ambas.

Arquivos de Dados de Entrada Especificados em um Arquivo. Processe todos os arquivos listados no arquivo especificado. O comando SPSSINC SPLIT DATASET pode produzir um arquivo no formato correto.

• Você deve escolher esta opção ou a opção anterior Dados de Entrada, mas não ambas.

Expressão de Procura. Insira uma expressão que avalie como true ou false utilizando os valores de variáveis em cada caso. Isso é equivalente a uma expressão que seria inserida em Data>Select Cases>If.

• Se a expressão não puder ser avaliado porque algumas variáveis não existem no arquivo sendo procurado, o caso não é selecionado.

• Use aspas simples se precisar incluir um valor de sequência de caracteres na expressão ou use duas aspas duplas. (A expressão toda ficará entre aspas duplas.)

Variáveis para Listar. Selecione uma ou mais variáveis para serem exibidas para casos correspondentes.

Processamento de Arquivo de Dados após Cada Passo. Escolha Fechar para fechar cada conjunto de dados após ele ser procurado.

Recursos adicionais

Execute o comando
SPSSINC PROCESS FILES /HELP.
para exibir a descrição toda da sintaxe. Este diálogo implementa um subconjunto dos recursos de SPSSINC PROCESS FILES. O diálogo Utilitários>Processar Arquivos de Dados suporta os recursos gerais deste comando.

Para utilizar um arquivo para especificar os arquivos que devem ser processados, crie um arquivo com um nome por linha, incluindo o caminho para o arquivo. O nome deve ser colocado entre aspas duplas ("). Qualquer coisa a seguir na linha será ignorada. Linhas em branco e linhas começando com # são ignoradas.

Requisitos

Este comando requer o Python Plug-in Programmability. O Plug-in pode ser localizado no CD do IBM SPSS Statistics, ou você pode fazer download dele no SPSS Developer Central (www.spss.com/devcentral).
